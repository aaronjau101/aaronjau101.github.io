Summary

Created by Manticore Games, Core is a 3D, Lua-based platform that developers can use to create games and metaverse experiences. I was apart of the Developer Relations team for two years at Manticore Games.

Contributions

I was tasked with engaging and educating a community of developers using the Core platform and Lua scripting. I hosted live events such as workshops, bootcamps, and competitions. I created technical documentation, tutorials, and API examples. I was a bridge of communication between developers and internal teams like Marketing, QA, and Product.
