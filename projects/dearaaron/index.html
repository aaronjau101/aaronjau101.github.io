Dear Aaron

JavaScript

Summary

Based off an old website named Peter Answers, Dear Aaron is sentient programming project capable of answering any questions.

Instructions

In order for Aaron to respond, you must type a question that starts with "Dear Aaron" and ends with a question mark "?".

Spoiler

The spoiler is that if type a dot "." at the start, then it will activate the secret mode. In secret mode, the program will continue to type "Dear Aaron, will you please answer this question:" whenever a key is pressed. Whatever is actually typed in secret mode will then be saved as the answer. Secret mode is deactivated when the dot "." is pressed again.
